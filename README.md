# 📊 Employee Data Analysis using Python

## 📌 Project Overview

This project performs comprehensive Exploratory Data Analysis (EDA) on an employee dataset using Python to uncover insights into workforce demographics, salary trends, departmental distribution, employee performance, education levels, and work modes.

The project follows a structured data analytics workflow that includes data preprocessing, statistical analysis, visualization, and automated report generation. An interactive Power BI dashboard is currently under development to further enhance data exploration and business insights.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualization using Matplotlib
- Automated PDF Report Generation
- Modular Python Code Structure
- Power BI Dashboard (In Progress)

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/vishwajeetjadhav0077-eng/Employee_EDA_Project.git
```

### Navigate to the Project

```bash
cd Employee_EDA_Project
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Project

```bash
python main.py
```
